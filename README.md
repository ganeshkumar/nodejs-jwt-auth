# nodejs-jwt-auth

### clone this repo


### Make sure yarn is installed in ur system by running
```sh
yarn --version
```

### install dependencies by running

```sh
yarn install
```


### goto project folder and run app
```sh
yarn run devStart
```

### open postman/ Advanced Rest Client
```sh
POST http://localhost:300/login

Add request body 
{
 "username": "FooBar"
}

Hit Submit

```
